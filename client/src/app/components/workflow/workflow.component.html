Auto Querying

Recommended

Auto Querying is a feature that automatically generates multiple relevant search queries based on the user's original input. Instead of relying solely on a single query, the system expands the input into several targeted queries to retrieve more comprehensive information from a knowledge base. This ensures more complete and accurate responses. You can customize how many queries are generated using the "Maximum queries" field. Additionally, the "Guidance" field allows you to provide extra context or instructions to steer how the queries should be constructed.
